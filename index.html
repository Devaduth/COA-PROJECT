Module 02

Pipeline Simulation

Simulate the classic IF → ID → EX → MEM → WB pipeline and observe cycle-by-cycle instruction flow.

IFInstruction Fetch — Read instruction from memory
IDInstruction Decode — Decode opcode & read registers
EXExecute — ALU operation or address calculation
MEMMemory Access — Read/write data memory
WBWrite Back — Write result to register file

Ideal CPI ≈ 1 once the pipeline is full. Data hazards cause stalls (bubbles) that increase CPI.

CPI_actual = CPI_ideal + Stall_cycles_per_instruction

Module 04

Amdahl's Law Speedup

Determine the theoretical speedup of a system when a fraction of the workload is enhanced.

Amdahl's Law Speedup = 1 / ((1 - P) + P / S)
Theoretical Max (S→∞) Speedup_max = 1 / (1 - P)

P = fraction that can be improved  |  S = speedup of enhanced portion
